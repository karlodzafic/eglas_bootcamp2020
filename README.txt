
  <----Upute za rad aplikacije--->

	Paketi i porgrami instalirani tokom izrade aplikacije:

		- node.js verzija: 12.10.0

		- npm     verzija: 6.10.3

		- yarn	  verzija: 1.17.3
		
		- Visual Studio Code



  <---- Upute za pokretanje aplikacije ---->

	S obzirom da se u gitignore nalazi node_modules, potrebno je napraviti par koraka prije pokretanje aplikacije:

			1. Terminal se mora nalaziti u direktorij eglas_bootcamp2020/igra tako da direktorij "igra" bude aktivna
			
			2. Sa naredbom "npm install" instalirava se direktorij node_modules sa svim potrebnim dadotekama
			
			3. Nakon instalacije sa naredbom "yarn start" se pokrene sama aplikacija na localhost:3000




  <---- Application instruction --->

	Packages and programs installed during application development:

		- node.js version: 12.10.0

		- npm version: 6.10.3

		- yarn version: 1.17.3

		- Visual Studio Code



   <---- Instructions for starting the application ---->

  	Since gitignore contains node_modules, you need to take a few steps before running the application:

			1. The terminal must be located in the eglas_bootcamp2020 / igra directory so that the "igra" directory is active

			2. The "npm install" command installs the node_modules directory with all the necessary files

			3. After installation with the command "yarn start" the application itself starts on localhost: 3000
